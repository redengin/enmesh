Firmware
================================================================================
Implementations to extend Meshtastic and Meshcore for use with enmesh.

This firmware will allow nodes to support both Meshtastic and MeshCore
simultaneously.


Supported Hardware
================================================================================
* Heltec (ESP32 based)
    * uses the ESP32 onboard WiFi to allow the device to use the enmesh bridge
        * users configure the device to connect to their local WiFi router
